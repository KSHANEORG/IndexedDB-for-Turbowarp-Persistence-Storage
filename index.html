Key Benefits

Persistent Storage

Enable your TurboWarp creations to remember user progress and data even after the browser/app/project is closed.

Offline Capabilities

Empower your projects to function even without an internet connection by leveraging local storage.

Improved Performance

Access and manage data efficiently, leading to smoother and more responsive TurboWarp experiences.

A very basic How To

1

Include the provided JavaScript extension in your TurboWarp project. (In [Add an Extension] --> [Custom extension] --> From file.
Don't forget to run it unsandboxed!

2

Utilize the simple API to store and retrieve data using IndexedDB.

3

Enjoy persistent and efficient local data management!
